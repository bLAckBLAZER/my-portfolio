First Python program:

Let's start your python journey with the classic "Hello World!" program, shall we? 

            
            print("Hello World!")

That's it! No kidding. You just created your first python program to display a message on the console. No class, no creation of objects, no semi-colons. It's that easy! Cool, right?

Let's create some variables:

Variables are integral part of any programming languages. They store any data you want and we can use them anywhere you want. If we want to change the value at some point, it saves us the trouble of making changes at every place the value or data is being used by just changing the value at variable assignment point. 

            
            name = "Omkar Jadhav"
            age = 22
            isGraduated = True
            height = 175.26

In Python, we also have various datatypes like integer, string, boolean, etc. But it saves you the trouble of restricting or pre-assigning a datatype to the variable while its declaration. Now you can use these variables wherever you want. 

                
            name = "Omkar Jadhav"
            age = 22

            newAge = age + 5

            print("Hello! My name is " + name)
            print("My current age is "+str(age) +
            ". After 5 years, I will be",newAge,"years old.")
Running above code will give you below output: 
                
            Hello! My name is Omkar Jadhav
            My current age is 22. After 5 years, I will be  27  years old.
